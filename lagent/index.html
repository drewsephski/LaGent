Getting Started with LaGent

Follow these simple steps to set up your first agent.

Step 1: Import Libraries 

from lagent import Agent

Step 2: Initialize the Agent 

my_agent = Agent(name="ChatBot", model="gpt-3.5-turbo")

Step 3: Define a Conversation Loop 

print("Welcome to ChatBot!")
while True:
    user_input = input("You: ")
    if user_input.lower() == 'exit':
        print("ChatBot: Goodbye!")
        break
    response = my_agent.respond(user_input)
    print(f"ChatBot: {response}")

Step 4: Run Your Agent 

python my_agent.py
